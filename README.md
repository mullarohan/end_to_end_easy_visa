# end_to_end_easy_visa

This is the end-to-end easy visa project, which aims to provide a seamless and efficient visa application process for travelers. The project includes a user-friendly interface, automated document processing, and real-time updates on visa application status. Our goal is to simplify the visa application process and enhance the overall experience for applicants.

## Features


### How to run this project
```bash
conda create -n easyvisa python=3.8 -y
```
```bash
conda activate easyvisa
```
```bash
pip install -r requirements.txt
```
```bash
python app.py
```
